# Rem Sigorta — Dijital Sigorta Acentesi

Yeni nesil dijital sigorta acentesi web sitesi. Koyu zeminli, profesyonel tasarım.

## Sayfalar

- **Ana Sayfa / Hakkımızda** (`index.html`) — Marka tanıtımı, ürünler, müşteri görüşleri
- **Atlas** (`atlas.html`) — AI maskot Atlas'ın hikayesi ve sosyal medya kartları
- **Teklif Al** (`teklif.html`) — 3 adımlı teklif formu

## Teknoloji

- Saf HTML, CSS, JavaScript (bağımlılık yok)
- Google Fonts: Syne + DM Sans
- Vercel static hosting
